Angular dialogs example

This demo shows the usage of the different types of dialog with angular. Note that dialogs are easy to use and customizable, where the available options are:

  • message
  • title
  • icon
  • button
  • reportButton
  • spin
  • closable
This demo uses Bootstrap 3.3.5, Angular 1.5.5 and Angular-UI-Bootstrap 2.0.1.
